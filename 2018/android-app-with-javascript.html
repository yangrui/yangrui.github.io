Android 客户端嵌入 JS 脚本

发表于 2018-09-26

APP 嵌入脚本能有效的降低重要功能硬编码带来的问题，尤其是新闻资讯这样依赖于多CP的应用，可以通过动态脚本在不更新应用的情况下升级功能，从而为应用程序提供灵活的扩展和定制功能。有很多中脚本语言选择：

  • Lua 是一种轻量小巧的脚本语言，其设计目的是为了嵌入应用程序中

  • JavaScript 是一种具有函数优先的轻量级解释型或即时编译型的编程语言

最有潜力发展最快的是 JavaScript，全栈且语法与 Java/C 更为相似

客户端转码需求

HTML -> DOM -> JSON

转码方案

  • Java 实现功能，使用 JSON 在后台配置规则
  • Java 提供脚本接口，使用 JavaScript 实现功能

纯函数的优点

  • 无状态 线程安全，不需要线程同步
  • 组合 纯函数相互调用组装起来的函数，还是纯函数
  • 缓存 应用程序或 Runtime 可以对纯函数的运算结果进行缓存，运算加快速度
