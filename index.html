The commit message style

发表于 2017-12-10

Header 是必需的，Body 可以省略。如果 Body 有内容，与 Heaer 之间必须有空行。为了清晰英文与中文之间添加空格，特别是 URL 两边与其他内容间需要添加空格。

type 用于说明 commit 的类别，只允许使用下面8个标识：

  • feat：新功能（feature）
  • fix：修补bug
  • docs：文档（documentation）
  • style： 格式（不影响代码运行的变动）
  • refactor：重构（即不是新增功能，也不是修改bug的代码变动）
  • test：增加测试
  • chore：构建过程或辅助工具的变动
  • res：资源（resource）

subject 是 commit 目的的简短描述，不超过50个字符。

  • 第一个字母小写
  • 以动词开头，比如“修改”
  • 结尾不加句号（.）分号（;）等符号

body 部分是对本次 commit 的详细描述，可以分成多行。应该说明代码变动的动机，以及与以前行为的对比。

给 Java 程序员的 JS 入门指南

发表于 2017-12-02

JavaScript 的特点

  • JavaScript是唯一浏览器可以识别的语言
  • 函数是顶级对象，闭包
  • 弱类型和原型继承：Java 程序员的噩梦
  • 对象和数组字面量：JSON 的灵感来源
  • JavaScript依赖于全局变量来进行连接

Java 和 JavaScript 就像雷锋和雷峰塔的区别

语法

  • var用来声明变量，否则会被认为是全局对象
  • 标识符可以使用_和$
  • 数字只有double类型，可以使用num|0转换成int
  • 字符串可以使用单引号或双引号包围，没有char类型
  • 假值：false, null, undefined, '', 0, NaN
  • 三元操作符a ? b : c 和 a && b || c
  • typeof运算符，数组和null结果是object

对象

JS 对象是可变键值集合（Keyed collections），键可以是任意字符串，值可以是除undefined之外任何值。如果键是合法的标识符，则不强制要求使用引号括住。

函数

JavaScript中的函数是一种特殊的对象，附加了函数上下文和实现函数行为的代码。闭包，函数可以定义在其他函数中，并可以访问父函数的变量和参数。

参数

函数调用会附加两个额外参数this和arguments。arguments包含所有参数，但不是真正的数组，没有数组方法。

调用

  • 方法：函数被保存为对象的属性，this调用时绑定
  • 函数：this被绑定到全局对象，而非外部函数this
  • 构造器：结合new前缀来调用
  • apply 或 call：改变this值

var不支持块级作用域，ES6 中可使用 let和const
